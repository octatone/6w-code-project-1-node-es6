# twit-scrip-js

I GETS 6W TWEETS

## Prerequisites

***requires latest node (not stable)***

install n
```
npm install -g n
```

load latest node
```
n latest
```

## Install Deps
```
npm install
```

## Run
```
KEY=your-twitter-app-consumer-key SECRET=your-twitter-app-consumer-secret node --harmony tweets.js
```
